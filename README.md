🏦 Bank Term Deposit Prediction

In this project, machine learning models are used to predict whether a customer will subscribe to a bank term deposit campaign.

Project Highlights
Cleaned and prepared marketing campaign data
Performed categorical encoding and feature preprocessing
Trained classification models including:
Logistic Regression
Random Forest
Evaluated model performance using:
Accuracy
F1-Score
Confusion Matrix
ROC Curve
Used SHAP Explainable AI to interpret predictions
Goal

To help banks identify potential customers and improve marketing campaign success rates.

🛍️ Customer Segmentation Analysis

This project analyzes customer purchasing behavior and groups customers into different segments using clustering techniques.

Project Highlights
Performed Exploratory Data Analysis (EDA)
Visualized spending patterns and annual income distribution
Applied K-Means Clustering for segmentation
Used the Elbow Method to find optimal clusters
Visualized clusters using PCA
Suggested marketing strategies for each customer group
Goal

To support personalized marketing and improve customer targeting strategies.

⚡ Energy Consumption Forecasting

This project focuses on forecasting household energy consumption using historical electricity usage data.

Project Highlights
Processed and prepared time-series data
Created time-based features for forecasting
Built forecasting models using:
ARIMA
Prophet
XGBoost
Compared model performance using MAE and RMSE
Generated forecast visualizations for trend analysis
Goal

To improve short-term energy usage prediction and support energy planning.

💳 Loan Default Risk Prediction

This project predicts loan default risk while considering the business impact of prediction errors.

Project Highlights
Cleaned and preprocessed financial data
Built predictive models using:
Logistic Regression
CatBoost
Performed business cost optimization
Analyzed false positives and false negatives
Identified important features affecting loan default
Goal

To help financial institutions reduce financial losses and improve credit risk management.
